Text
1. Strong. The (strong) tag is used to give importance to text, indicating that it is of strong significance. By default, text within this tag is typically displayed in bold. strong
2. Emphasized. The (em) tag is used for emphasized text. This usually renders text in italics, indicating stress or emphasis. emphasized
3. Semantic. The (i) tag is used for text that is in an alternate voice or mood. It often represents italicized text, which is used for semantic purposes, like foreign words or technical terms. semantic
4. Strikethrough. The (s) tag represents text that is no longer accurate or relevant, typically shown with a strikethrough. strikethrough
5. Abbreviation. The (abbr) tag is used to indicate an abbreviation or acronym. The title attribute provides the full form. abbr
6. Mark. The (mark) tag is used to highlight text that has relevance to the content, typically shown with a yellow background. mark
7.Underline. The (u) tag is used for underlined text. It is often used for text that should be highlighted or emphasized in a specific way. underline
8.Superscript. The (sup) tag is used to display text as superscript, which is typically raised above the normal line of text. sup
9. Subscript. The (sub) tag is used to display text as subscript, which is typically lowered below the normal line of text. sub
10. Inline Quote. The (q) tag is used for short inline quotations. Browsers typically add quotation marks around the content. inline quote
11. Time. The (time) tag is used to represent a specific period in time. The datetime attribute provides a machine-readable format. May 18
12. Block Quote. The (blockquote) tag is used for longer quotations, usually displayed as a block of text that is indented.
block quote

13. Term Definition. The (df) tag is used to define a term. The first occurrence of the term in a document is typically wrapped in this tag. term
14. Citation. The (cite) tag is used to reference the title of a creative work, such as a book, movie, or article. citation
15. Data. The (data) tag is used to link the content to a machine-readable value. The value attribute provides the underlying data. data
